TkCVS SVN

Mission

  • To add Subversion support to TkCVS, which has a user base as a cross-platform revision-control GUI.  It runs on Unix/Linux, Windows, and MacOSX with the "native" Tk on each platform.  TkCVS was written by D. Elson many years ago, but there's life in it yet.


Ground Rules

  • The project is written in plain old Tcl/Tk.  That makes it easy to support users across different platforms, and to distribute as a self-contained .app on MacOSX.


How You Can Help

  • I don't use Subversion in an employment context, so I'm just starting to learn it.  Patches are welcome if I've got the usage model wrong, or I'm ignoring important functionality.
  • Is it possible to do a good branch/tag diagram a la CVS?  If you know how, you'll have a grateful world.
